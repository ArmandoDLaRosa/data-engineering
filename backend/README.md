# Backend

Requirements


## Setup

From root (backend/) in cmd, run:

```sh
$ pipenv shell
```

```sh
$ pip3 install -r requirements.txt
```

```sh
$ pip freeze > requirements.txt
```

## API

if you have run  `pipenv shell`, then from root (backend/), run:

```sh
$ flask run
```

## Database

If you want to create the table, run:

```sh
$ python3.8 database.py
```
